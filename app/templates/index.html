About:
  • Welcome to this Healthcare Records Manager application.

    This platform is designed to securely manage and store patient healthcare records, for use in stroke prediction AI modelling, to help try and find new ways to improve outcomes for participating patients.

    The data collected on each patient is strictly confidential and is protected in accordance with both GDPR regulations and NHS data security standards. Along with strong security developed techniques to ensure even in the event of a data breach, patient data remains protected.

  • Please use the login or registration buttons bellow, or at the top right of the page to access this application. If you are a new user, you will need a valid patient ID number and email address to complete the registration process.

    Thank you for using this application, have a great day!
Compliance:
  • This application is complient with GDPR regulations, NHS data security standards and the "Open Worldwide Application Security Project" (OWASP) guidelines.

    Including multiple security testing techniques inlcuding with OWASP's ZAP toolkit.
